## Overview / 概览

This domain covers the design of LLM user experiences, with a focus on how AI output formats are evolving from plain text toward richer, more visually efficient representations. Key topics include output format progression (text → markdown → HTML → interactive media), multimodal input/output, and the broader human-AI communication bandwidth problem.

本领域关注 LLM 用户体验设计，重点是 AI 输出格式如何从纯文本向更丰富、视觉效率更高的方向演进。核心话题包括输出格式递进（文本 → Markdown → HTML → 交互式媒体）、多模态输入输出，以及更广泛的人机通信带宽问题。
